В директории /tests располагаются файлы с тестами 
В conftest.py находятся 3 фикстуры: автризация, открытия страницы мои питомцы, подсчет питомцев 
Для выполнения скрипта необходимо создать файл .env в корневой директории проекта со следующими переменными:


email=ваш_email@example.com
password=ваш_пароль

Важно:
Используйте данные зарегистрированного пользователя
Файл .env добавлен в .gitignore для безопасности
